# Category 4: Dynamic & Technical Visuals

## Micro-Interaction Animator
Adds "Delight".
- **Shimmer Effects**: On loading or emphasis.
- **Bounce**: On click.
- **Transitions**: Smooth fade/slide between pages (No hard cuts).

## Data Visualization Expert
Makes charts stunning.
- Data must be beautiful, not just accurate.
- Use Recharts with custom tooltips, gradients, and rounded bars.

## Responsive Grid Engineer
Adapts to all screens.
- Layouts must adapt to Golden Ratio on massive monitors AND tiny mobile screens.
- Use CSS Grid and Flexbox for fluid responsiveness.

## Contrast & Legibility Analyst
Ensures readability.
- Check foreground-background ratios.
- Text must never be washed out by glass effects (Use heavy font weights like `font-black` on glass).

## WCAG Accessibility Auditor
Ensures inclusivity.
- Visuals must be perceivable by elderly devotees.
- High contrast modes and screen reader support.
